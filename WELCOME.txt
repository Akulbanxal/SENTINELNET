   📖 Start with: README.md
   ⚡ Quick setup: QUICKSTART.md
   🗺️  Structure: PROJECT_STRUCTURE.md

2️⃣  Set Up Environment
   cp .env.example .env
   # Add your API keys:
   # - PRIVATE_KEY (MetaMask)
   # - ALCHEMY_API_KEY (alchemy.com)
   # - OPENAI_API_KEY (platform.openai.com)

3️⃣  Run Automated Setup
   ./setup.sh
   # This will:
   # - Install all dependencies
   # - Compile contracts
   # - Guide you through deployment

4️⃣  Deploy & Register
   cd contracts && npm run deploy:sepolia
   cd ../agents && npm run register:agents

5️⃣  Start Everything
   # Terminal 1:
   cd backend && npm run dev
   
   # Terminal 2:
   cd frontend && npm run dev
   
   # Terminal 3:
   ./scripts/start-agents.sh

6️⃣  Access Dashboard
   🌐 Open: http://localhost:3000
   ✨ Watch the magic happen!

🎯 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. TraderAgent discovers new token 🔍
2. Queries AgentMarketplace for verifiers 📋
3. Creates escrow job with payment 💰
4. Verification agents analyze token in parallel 🔬
   → SecurityAgent: Checks for vulnerabilities
   → LiquidityAgent: Evaluates liquidity risks
   → TokenomicsAgent: Reviews token economics
5. Agents submit reports on-chain 📄
6. TraderAgent aggregates risk scores 📊
7. If safe, executes simulated trade ✅
8. Dashboard updates in real-time 🔄
